MERGING

Merging New Changes To Existing Repository From a New Machine

$ git init

$ remote -v

$ git add .

$ git commit -m

$ git config --global.user.email "wsxrfv9090davyyu@gmail.com"

$ git config --global user.name "Davy"

$ git remote add origin https://github.com/wsxrfv9090/Test.git

$ git pull origin master

$ git push origin master

Ps: Sometimes it may need you to authenticate, if so, please use the personal access token generated in Github

Ps: Using pull command will end up telling you some errors and conflicts about mayber history and other stuff, after resolving the conflicts, git will Resolve any conflicts, if there are any. Git will provide instructions on how to handle conflicts.
